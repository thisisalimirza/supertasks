Frequently asked.

Is it really free? What’s the catch?+
No catch. SuperTasks is free software. No Pro tier, no trial, no feature gating. Your data lives on your Mac — there are no servers to run, which means there’s nothing to charge you for. If it earns your support via Ko-fi, great. If not, that’s fine too.
Where is my data stored?+
In a local SQLite database on your Mac — typically at ~/Library/Application Support/SuperTasks/tasks.db. Open it with TablePlus, DB Browser for SQLite, or script against it however you like. It’s your data.
Does it sync with iPhone?+
Not yet. iOS sync is on the roadmap. Sign up for the newsletter below to get notified when it ships. Your SQLite file isn’t going anywhere in the meantime — it’ll migrate cleanly when sync arrives.
What if you stop developing it?+
Your data is in a standard SQLite file. It doesn’t disappear if the app does. Open it with any SQLite browser, export it, or build a migration script. No lock-in by design — that’s a feature, not a footnote.
Does it work offline?+
100%, always. There’s no network dependency at all. The core app works in airplane mode, on a flight, in a basement. Everywhere.
Do I need to be a developer to use it?+
Not at all. The J/K shortcuts for moving up and down take about five minutes to get used to — after that, everything clicks. If you use keyboard shortcuts in any app, you can use SuperTasks. The only requirement is a Mac.
How is it different from Things 3?+
Things 3 is a genuinely polished app — it’s beautiful. The difference is philosophy. Things 3 is designed around visual hierarchy and mouse navigation. SuperTasks is built around the keyboard: Vim-style movement, G-chord view switching, a command palette for every action. If you type faster than you click, you’ll feel the difference within minutes. Also: SuperTasks is free and Things 3 costs $49.99.
